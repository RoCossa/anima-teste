# AnimaWebAPI

# Configurações
- Utilizar o SQL Server Express LocalDB para o banco de dados.
- Aplicar as migrations para criar o banco:
    - A partir da linha de comando na pasta do projeto *Web.APi.Infrastructure* execute: <code>Web.Api.Infrastructure>**dotnet ef database update**</code>


# Visual Studio
Abra o arquivo <code>AnimaWebAPI.sln</code> e rode.

# VSCode
Abra a pasta src e pressione F5

# Acessa o Swagger
http://localhost:5000/swagger
